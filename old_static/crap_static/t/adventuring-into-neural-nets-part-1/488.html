Let‚Äôs write up a quick function for the simplest kind of neural network. One dubbed the ‚Äúperceptron‚Äù. Let‚Äôs take a quick look first.

So, let‚Äôs take this progressively. The inputs are gonna be real numbers. Each is muliplied by it‚Äôs weight, then they are summed for the final outcome in the end layer.     return _in1*_w1+_in2*_w2

Now, let‚Äôs briefly generalize this situation and simplify for N inputs :).

def gen_perceptron(_in,_w):
    ret = 0
    for i in range(len(_in)):
        ret = ret + _in[i]*_w[i]
    return ret/abs(ret)
