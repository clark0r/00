TIG - A git repository browser for your terminal!

Linux
kowalski March 12, 2017, 7:28am 1

Text mode interface for git.

Tig is an ncurses-based text-mode interface for git. It functions mainly as a Git repository browser, but can also assist in staging changes for commit at chunk level and act as a pager for output from various Git commands.

GitHub

jonas/tig

Text-mode interface for git. Contribute to jonas/tig development by creating an account on GitHub.

The docs are at https://jonas.github.io/tig
