|   TIME    | BV  | C1  | C2  | C3  | C4  | ... |
+-----------+-----+-----+-----+-----+-----+-----+

It seems to me that using bayesian inference to model probable outcomes would be ideal, but am looking for others thoughts before I set off on this project.
