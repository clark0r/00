Hey everyone! I just thought I would share this little technique, because it was unknown to me after a year of programming experience!

int a = 5, b = 6;

// swapping now:

a = a + b; 
// a = 11

b = a - b; 
// b = a + b - b
// b = a
// b = 5

a = a - b; 
// a = a + b - (a + b - b)
// a = b
// a = 6
