Hi 0x00’ers! We had a request to make a Linux category. Since Linux is so apt to most hackers’ workflows, it would be silly to not support it here.

In response we now have two new categories:

  • https://0x00sec.org/c/reference-database/linux - Which is to post cheatsheets and other things perhaps needed for later reference

  • https://0x00sec.org/c/How-to/linux - Which is for Tutorials and How-to’s related to Linux such as @L3akM3-0day’s tutorial on SSH tunneling.

Have fun
